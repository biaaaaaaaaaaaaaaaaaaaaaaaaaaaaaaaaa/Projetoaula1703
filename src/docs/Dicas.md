1º Abrir o VSCODE
2º No terminal digitar: cd desktop
3º No terminal dentro do desktop: mkdir *nome da pasta*
4º No terminal: cd *nome da pasta*
5º Dentro da pasta: code .
6º No terminal da pasta: touch index.html
7º No terminal; mkdir src
8] No terminal: cd src
9º No terminal; mkdir css
10º No terminal: mkdir js
11} No terminal: cd css
12 No terminal: touch estilo.css
13º No terminal: cd ..
14º No terminal; cd js
15º No terminal: touch script.js
16º No terminal: cd ..
17º No terminal: cd .. até chegar na raiz do projeto
18º No terminal: git init
19º se estiver na fiap: git config --global user.name "Seu primeiro nome"
20º se estiver na fiap: git config --global user.email Seu email
21º No terminal: git status
22º Se estiver vermelho: git add .
23º Se estiver verde: gt commit -m "Mensagem do que você fez"
24º Entrar no Github em repositórios, criar um novo repositório, colocar o nome e deixar público, clicar no botão criar repositório (DICA DE OURO- NUNCA TRADUZA A PÁGINA DO GITHUB)
25º No bloco que estiver as 3 linhas: 
 1º linha- Conecta seu gitbash com o seu github
 2º linha- Muda o nome master para main
 3º linha- Pega tudo que estiver no projeto e sobe para o github
26º Caso você esteja em outro pc: no seu repositório no botão verde(code), copie o link 
27º Abra o seu VSCODE no terminal cd desktop : git clone (copie o link), isso vai clonar seu projeto e pode continuar de onde parou.
28º Se modificar se código faça todo o processo go 
    git status
    git add .
    git commit -m ""
    e para subir alterações digite: git push
